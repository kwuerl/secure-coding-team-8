\begin{longtable}[l]{ p{2.3cm} | p{.79\linewidth} }\hline
	& \textbf{Online Banking} \\ \hline
	\textbf{Observation} & Online Banking does not use HTTPS and therefore does not implement HSTS. \\
	\textbf{Discovery} & Accessing the site with \code{https://IP\_ADDRESS/Online Banking/} shows that HTTPS is not used. Furthermore, using the command \code{curl -s -D- http://\allowbreak IP\_ADDRESS/Online Banking/ | grep Strict} did not yield any results and therefore the header for \code{Strict-Transport-Security} is not set. \\
	\textbf{Likelihood} & N/A \\
	\textbf{Impact} & N/A \\
    \textbf{Recommen\-dations} & Use and enforce HTTPS for a secure communication. If using HTTPS, the HSTS header for \code{Strict-Transport-Security} has to be set to \code{max-age=\allowbreak 60000; includeSubDomains}. \\ \hline
	\textbf{CVSS} & N/A \\ \hline
\end{longtable}

\subsubsection{Comparison}
Online Banking does not use HTTPS and therefore the header for \code{Strict-Transport-\allowbreak Security} is not set. SecureBank enforces HTTPS and the header for \code{Strict-Transport-\allowbreak Security} is set to \code{max-age=\allowbreak 60000; includeSubDomains}.
